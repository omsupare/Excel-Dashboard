📊 Sales Dashboard (Advanced Excel Project)
📌 #Overview

This project is an interactive Sales Dashboard built using Microsoft Excel. It provides a comprehensive view of sales performance across different regions, cities, product categories, and payment modes. The dashboard helps in analyzing key business metrics and supports data-driven decision-making.

🎯#Objective

The main objective of this project is to:

Analyze sales data efficiently

Track business performance using KPIs

Provide interactive filtering for better insights

Visualize complex data in a simple and understandable format

🛠️##Tools & Technologies Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers & Timeline

Data Cleaning Techniques

Conditional Formatting

📂 #Dataset Description

The dataset includes the following fields:

Order Date

Region (Central, East, North, South, West)

City (e.g., Nagpur, Bhopal, Mumbai, etc.)

Product Category & Sub-category

Sales Channel (Online/Offline)

Payment Mode (Card, COD, UPI, Wallet)

Delivery Status (Delivered, Cancelled, Returned, In-Transit)

Sales & Profit

📈 #Dashboard Features
🔹 Key Performance Indicators (KPIs)

Total Sales: ₹56,21,721.01

Total Profit: ₹13,15,301.97

🔹 Interactive Filters (Slicers)

Sales Channel

Payment Mode

Delivery Status

Region

City

Product Category & Sub-category

Timeline (Date filter)

🔹 #Visualizations Included

Donut Chart for Payment Mode distribution

Pie Chart for Online vs Offline Sales

Tree Map for Delivery Status

Bar Chart for Product Sub-category performance

Line/Area Chart for City-wise trends

Region-wise sales comparison

💡 #Key Insights

Majority of sales come from Online channels

Payment distribution is almost evenly spread across modes

Certain product categories contribute significantly more revenue

Delivery performance shows areas for operational improvement (returns, cancellations)

⚙️ #Key Functionalities

Dynamic filtering using slicers

Automatic updates of charts based on selection

Clean and structured dashboard layout

User-friendly interface for non-technical users

🚀 #How to Use

Open the Excel file

Use slicers (filters) to select:

Region, City, Product Category, etc.

Analyze updated charts and KPIs dynamically

Gain insights based on filtered data
